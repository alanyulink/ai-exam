### 第18题 【单选题】
**题目：** 下列哪一个是在目标检测中用于进行物体定位的方法?

- A. 选择性搜索
- B. 边界框回归器
- C. 候选区域网络
- D. Fast R-CNN

**正确答案：** B

**【解析】**
**A. 选择性搜索（Selective Search）**
- 用于生成候选区域
- 是一种区域提议方法
- 在R-CNN和Fast R-CNN中使用
- 作用是"找哪里可能有物体"，而不是精确的定位
- 生成的候选框比较粗糙
**B. 边界框回归器（Bounding Box Regressor）**
- 用于精确调整边界框的位置和大小
- 对候选框进行微调，使其更接近真实物体
- **直接用于物体定位**
- 输出边界框的偏移量，精修位置
**C. 候选区域网络（RPN）**
- Region Proposal Network
- Faster R-CNN中提出
- 用于生成候选区域（候选框）
- 虽然也涉及边界框，但其主要作用是生成候选，而不是最终的精确定位
- RPN生成的候选框还需要后续的检测头进一步精修
**D. Fast R-CNN**
- 是一个完整的目标检测模型
- 不是一个具体的"方法"
- 包含了多个组件（特征提取、RoI Pooling、分类、回归）

### 第24题 【单选题】
**题目：** 在使用大模型处理敏感数据时，以下哪种部署方式最安全？

- A. 本地私有部署
- B. 公共云API
- C. 在线网页版
- D. 移动应用内置

**正确答案：** A

**【解析】**
处理敏感数据时，数据安全和隐私保护是首要考虑因素。
**各部署方式安全性对比：**
**1. 本地私有部署（最安全）**
- 模型和数据都在企业/用户自己的服务器上
- 数据不会流出企业网络边界

### 第1题 【多选题】
**题目：** 以下哪些属于典型的局部特征点检测方法？

- A. Harris角点检测
- B. FAST特征检测
- C. Canny边缘检测
- D. SIFT特征检测

**正确答案：** ABD

**【解析】**
局部特征点检测（Local Feature Point Detection）是计算机视觉的基础技术，用于检测图像中具有辨识度的关键点（Interest Points / Keypoints）。典型方法包括：
**1. Harris角点检测（Harris Corner Detector）**：
- 1988年提出，经典的角点检测算法
- 原理：基于图像灰度的二阶矩矩阵，分析局部区域的灰度变化
- 特点：简单有效，对旋转、光照变化鲁棒
- 局限：不具有尺度不变性
**2. FAST特征检测（Features from Accelerated Segment Test）**：
- 2006年提出，高效的角点检测算法
- 原理：比较中心像素与周围16个像素的灰度差异，快速判断是否为角点
- 特点：速度极快，适合实时应用
- 常与BRIEF描述子结合使用（ORB算法）
**3. SIFT特征检测（Scale-Invariant Feature Transform）**：
- 1999年提出，2004年完善，最经典的局部特征算法
- 原理：在多尺度空间中检测极值点，精确定位关键点位置和尺度
- 特点：具有尺度不变性、旋转不变性、对光照变化鲁棒
- 包含检测和描述两个部分：SIFT关键点 + SIFT描述子
- 被誉为"特征检测之王"
**4. 其他重要方法**：
- SURF：SIFT的加速版本
- ORB：FAST + BRIEF的改进，速度快且免费
- KAZE/AKAZE：基于非线性尺度空间
选项C（Canny边缘检测）是边缘检测算法，不是特征点检测方法。边缘检测的目标是找到图像中灰度突变的边缘线条，而特征点检测的目标是找到具有辨识度的离散关键点。

- B. 图像数据未归一化也可以直接用于模型的训练
- C. 在深度学习中，通常使用训练集中图像的均值和标准差进行归一化操作，以保证训练和测试数据的处理方式一致
- D. 将图像的像素值归一化到 [-1, 1] 的范围，通过减去均值并除以标准差的操作，以消除输入数据全为正时可能出现的影响

**正确答案：** ACD

- 选项A正确
**2. 标准化（Standardization）/零均值归一化**：
- 将数据转换为均值为0、标准差为1的分布
- 方法：(x - mean) / std
- 结果分布在[-1, 1]附近（具体范围取决于数据分布）
- 目的：
- 消除不同特征尺度的影响
- 输入全为正时，反向传播可能导致梯度方向不一致，零均值可以改善

- 原因：
- 保证数据处理方式的一致性
- 避免数据泄露（不能使用测试集的统计信息）
- 模拟真实部署场景（部署时不知道整体数据分布）
- 选项C正确
选项B的说法有争议。虽然理论上不归一化也可以运行训练，但：
- 训练会变得不稳定，收敛困难
- 梯度可能出现爆炸或消失
- 模型性能会显著下降

**题目：** 线性滤波是用于消除图像中的噪声的一种方法。

- 对
- 错

**正确答案：** 对

**【解析】**
线性滤波是图像处理中最基础也是最常用的去噪方法之一：
  1. **什么是线性滤波**：
     - 线性滤波是指滤波输出是输入像素的线性组合
     - 数学上表示为输入图像与卷积核（也叫滤波器、掩模）的卷积运算

  2. **常见的线性滤波去噪方法**：
     - **均值滤波（Mean Filter）**：用邻域像素的平均值代替中心像素值，能够平滑噪声，但会模糊边缘
     - **高斯滤波（Gaussian Filter）**：使用高斯函数作为卷积核，对中心像素赋予更高权重，去噪效果比均值滤波好，边缘模糊程度相对较轻

**题目：** fastText 使用词内的n-gram信息(subword n-gram information)的原因是因为它能更好地处理拼写错误、罕见词和新词。

- 对
- 错

3. **为什么能处理罕见词和新词**：

### 第8题 【判断题】
**题目：** 反卷积是卷积的逆运算，用于将数据从低维还原到高维。

- 对
- 错

**正确答案：** 错

**【解析】**
2. **为什么不是逆运算**：
- 卷积操作会丢失信息（降采样后无法完全恢复原始信息）
- 反卷积只能恢复空间尺寸，但不能恢复原始的精确值
- 反卷积是一个"近似"的逆变换，学习到的是上采样的权重
- 数学上，它对应卷积矩阵的转置矩阵相乘，而不是逆矩阵相乘
- 上采样：将小尺寸特征图放大为大尺寸特征图
- 在语义分割中，将编码器提取的低分辨率特征恢复到原始图像尺寸
- 在生成模型中，从低维隐向量生成高分辨率图像
- "反卷积"这个名称容易引起误解，更准确的叫法是"转置卷积"
- 也常被称为"上卷积"（up-convolution）
- 在有些文献中也叫"后向传播的卷积"
- **双线性插值**：不需要学习，速度快，但可能产生模糊结果
- **最近邻插值**：最简单，计算快，但质量差
- **反卷积**：可学习参数，能够学习到更好的上采样方式
- **PixelShuffle**：另一种可学习的上采样方法
- 语义分割（如FCN、U-Net）
- 图像生成（如GAN的生成器）
- 超分辨率重建
- 目标检测中的特征金字塔
